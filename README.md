# 📅 Reservation App (Calendar付き)

シンプルな予約アプリです。
チェックイン・チェックアウトの日付をカレンダーから選択できるようにしています。
帝国ホテルの予約画面を意識してデザインしています。

---

## 🚀 機能

* カレンダー表示（前月・次月の切り替え）
* 日付選択機能（チェックイン / チェックアウト）
* 無効日（過去日）の制御
* フォーム入力

---

## 🛠 使用技術

* HTML
* CSS
* JavaScript（Vanilla JS）
* php（Vanilla php）


---

## 💡 工夫した点

* `Date`オブジェクトを使って月末日を動的に取得
* 配列とオブジェクトで日付データを管理
* `isToday` / `isDisabled`などのフラグでUI制御
* DOM操作でカレンダーを動的に生成
